## 💡 Features

- Auto Fire on Crosshair Lock  
- Adjustable Delay Curve  
- Scoped Mode Toggle  
- Stream Overlay Safe  
- Configurable Keybinds

---

## 🛠️ Setup

1. Start Apex  
2. Run `Loader.exe`  
3. Choose trigger profile
